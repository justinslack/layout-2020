Masonry

Masonry is a layout method where one axis uses a normal grid layout, usually columns, and the other a masonry layout. On the masonry axis, rather than sticking to a strict grid with gaps being left after shorter items, the items in the following completely fill the gaps. (See Masonry layout on MDN for the full specification.

Note that Masonry is only availbale in Firefox behind a flag. Go to about:config and set the layout.css.grid-template-masonry-value.enabled flag to true.

Masonry columns

Apply block-size to children


.masonry {
  display: grid;
  gap: 0;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  grid-template-rows: masonry;
}

Masonry rows

Apply inline-size to children


.masonry {
  display: grid;
  gap: 0;
  grid-template-columns: masonry;
  grid-template-rows: repeat(3, 100px);
}
