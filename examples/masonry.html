Masonry

The Box Alignment Specification details how alignment works in various layout methods. The specification identifies the following alignment methods:

Masonry columns

Apply block-size to children


.masonry {
  display: grid;
  gap: 0;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  grid-template-rows: masonry;
}

Masonry rows

Apply inline-size to children


.masonry {
  display: grid;
  gap: 0;
  grid-template-columns: masonry;
  grid-template-rows: repeat(3, 100px);
}
